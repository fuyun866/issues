1. git init 
   - 初始化git
2. git add README.md
   - 添加到暂存区
   - 正常git add . 就可以了
3. git commit -m "first commit"
   - 将暂存区内容添加到本地仓库中。
   - 后面的引号内容是提交每个文件夹后面的说明
4. git branch -M main
   - 建立一个main分支
5. git remote add origin git@github.com:fuyun866/-.git
   -连接自己的远程仓库
6. git push -u origin main
   - 真正的提交
